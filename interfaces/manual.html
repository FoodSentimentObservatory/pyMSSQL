A brief walk-though how to use the system

  • 1. Get to the home page

    There you'll see three groups of input to specify:

    • Search type (currently only the middle option available)
    • Choose sprint - select a discourse from the dropdown menu. Once you click on it, you will also see some additional information about the search to help you decide.

  • 2. Specify keyword groups

    The results will need to contain either all words from one group or from another

    When typing words, you'll get autosuggested keywords and phrases from the selected search, however, you can pick your own as well.

    You can also specify a time range for your search. By default the time range is the whole search period for the selected discourse.

    Note: Please keep in mind that each tweet has a limited amount of characters, so the more words a group has, the less likely it is to return a lot of results

  • 3. Examine results

    Each group results come with their own basic statistics

    If you click on a word from the frequent word section, you'll get all the tweets from the current group results containing that word. You can continue doing this until you get to a point where the frequent word generated, show the same results. This means you've reached the end of the rabbit hole and you've ended up with one tweet or a group of retweets.
